Execute code.

object Main {
    // do something

    def main(args: Array[String]) {
        // do something
    }
}

Compile with: $ scalac filename.scala

Run with: $ scala filename

Handle exceptions.

try {
    // do something
}
catch {
    case zero: ArithmeticException => println("This is an arithmetic error: " + zero)

    case e: Throwable => println("Error: " + e)
}
finally {
    // do something
}

Add build system to build and run in Sublime Text.

{
    "cmd": ["scala", "$file"],
    "path": "$PATH:/usr/local/bin",
    "file_regex": "^(.+):(\\d+): .+: (.+)",
    "selector": "source.scala"
}
